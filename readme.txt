Qocai REDeath 2023

-------------------------

Игра REDeath - тот еще шутер. Выполнен в хайповой тематике зомби-апокалепсиса. Вам предстоит делать зачистки на бесконечном количестве локаций. Играйте так, как захочется вам! А помогут в этом ваши любимые треки, а также уникальная технология генерации ландшафтов по музыке

& Написать про выбор уровня, особенности подгрузки локаций к трекам, управление и стреляние, сохранения и их загрузку
Для выбора изображения требуется перейти по меню: Файл >> Открыть. Принимаются изображения с расширениями: .jpg, .png, .gif, .bmp, .tiff, .jp2
Сохранение: Файл >> Сохранить
Выход можно совершить нажатием на крестик, при этом прогресс не сохраняется

Отображаются слои в виде вкладок, где слева самые нижние, а справа - верхние. Для добавления слоя нажмите "+ Слой", для удаления - на крестик нужного слоя. Чтобы 
переименовать слой, необходимо дважды щелкнуть по нужному слою, ввести новое имя и нажать Enter. Для перемещения слоя вверх или вниз нажмите на него и, 
удерживая переместите его. Можно очистить слой, нажав на кнопку "Очистить слой"

Для добавления своей музыки нужно в папку music/mp3 загрузить трек в этом формате, затем в папке music/wav создать копию этого трека, конвертированную в этот формат, с одним каналом, 16 бит. Содержимое первой папки используется для воспроизведения во время игры, а второй - для генерации на основе них ландшафтов

Настройки кисти:
# Размер - в пикселях, от 0 до 1000
# Прозрачность - от 0 до 255
# Цвет - можно записать в шестнадцатиричном формате RGB или выбрать визуально

Вашему вниманию представлено 4 вида зомби:
# Зомби-модник - хлипкая фигура в совокупности с отсутствием базовых навыков борьбы. В мирное время любил ходить по бутикам, иногда ходил в фитнес. По умолчанию имеет 1 hp и наносит 1 урона/с
# Зомби- - часто ходил в качалку
# Зомби-
# Зомби-военный - самый трудный для убийства, ведь недаром его тренировали. Правда во время зомби-апокалепсиса все пошло не по плану, и некоторые служащие были захвачены. По умолчанию он имеет 7 hp и наносит 4 урона/с

Смотрите на мир так, как удобно вам! Вид можно регулировать клавишами w, a, s, d. При одном нажатии камера передвигается, для остановки нажмите противоположную клавишу

Изюминкой моей программы является изменение времени суток, со специально подобранной цветовой палитрой закатов и рассветов. Наслаждитесь эстетикой постапокалептического мира!
Вы также можете кастомизировать настройки, переписав их в файле settings.txt на свой лад. Через пробел указываются ...
